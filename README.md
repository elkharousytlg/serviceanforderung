# Serviceanforderung (Service Request DB)

A lightweight, mobile-responsive web application for managing service requests and part numbers.

## Features
* **Offline Storage:** Uses browser-based IndexedDB (no server required).
* **Search & Filter:** Real-time filtering across positions, article numbers, and descriptions.
* **Mobile Optimized:** Designed with a "Mobile-First" approach for use on the shop floor.
* **PWA Ready:** Can be added to the home screen on iOS and Android.

## Deployment
This project is hosted via GitHub Pages. 

## Technical Details
* **Language:** HTML5, CSS3, Vanilla JavaScript.
* **Database:** IndexedDB API.
* **Fonts:** IBM Plex Mono & Sans.
